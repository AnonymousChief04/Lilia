lia.module

Core library that manages module loading behaviors.

Functions

lia.module.load(uniqueID, path, isSingleFile, variable)
Internal

This is an internal function! You are able to use it, but you risk unintended side effects if used incorrectly.

Loads a module into the system. This function loads a module into the system, making its functionality available. It sets up the module environment, including defining globals and loading necessary files.

Parameters

  • uniqueID string

    string The unique identifier of the module.

  • path string

    string The path to the module.

  • isSingleFile boolean

    boolean Specifies if the module is contained in a single file.

  • variable string

    string The variable name to assign the module to.
